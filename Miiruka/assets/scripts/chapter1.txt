# Capitulo 1 - ejemplo de guion por eventos
# Dialogo bilingue:
# [personaje][Jouktai][habla][es:Hola][way:Wanee]
# Si solo hay un texto, se usa como fallback para ambos idiomas.
#
# Ordenes disponibles:
# [escena][Nombre]
# [label][Nombre] o [etiqueta][Nombre]
# [goto][Nombre] o [ir_a][Nombre]
# [cambiar_escena][NombreScene]
# [plano][sol_hacia_abajo]
# [personaje][Nombre][entra][izquierda|derecha]
# [personaje][Nombre][emocion][feliz|triste|duda]
# [personaje][Nombre][habla][es:...][way:...]
# [minijuego][id][respuesta1][respuesta2]
# [if][id][respuesta1][...accion...]
# [mostrar][imagen][textureKey]
# [camara][subir|bajar][distancia][duracion]
# [esperar][ms]
# [fin_escena]

[escena][Inicio]
[plano][sol_hacia_abajo]
[personaje][Jouktai][entra][izquierda]
[personaje][Jouktai][habla][Buenos dias, mi nombre es Jouktai. Estoy yendo a buscar agua para llenar mi cantaro.]
[personaje][Jouktai][habla][es:Me ayudas a girar la llave del agua?][way:wayuuuu]
[label][intento_llave]
[minijuego][girar_llave][respuesta1][respuesta2]
[if][girar_llave][respuesta1][personaje][Jouktai][emocion][duda][habla][Que raro, ayer el tanque estaba lleno.]
[if][girar_llave][respuesta1][personaje][Jouktai][habla][En nuestra comunidad usamos un molino para bombear agua. Ire a revisarlo.]
[if][girar_llave][respuesta2][personaje][Jouktai][emocion][triste][habla][No salio agua. Intentemos otra vez.]
[if][girar_llave][respuesta2][goto][intento_llave]
[cambiar_escena][Chp1_scn2]

[escena][Desarrollo]
[personaje][Jouktai][entra][izquierda]
[personaje][Jouktai][habla][Hoy Kai esta practicando la Yonna con mi tio. Acompaname a buscarlo.]
[personaje][Jouktai][habla][En mi comunidad el agua nos da vida. Cada gota cuenta.]
[personaje][Kai][entra][derecha]
[personaje][Kai][habla][Hola Jouktai, estaba practicando la Yonna con mi papa.]
[personaje][Jouktai][habla][Necesito tu ayuda, no se donde esta el molino.]
[personaje][Kai][habla][Debe estar en un lugar abierto, lejos de casas, arboles y cableado.]
[minijuego][encontrar_molino][respuesta1][respuesta2]
[if][encontrar_molino][respuesta1][personaje][Jouktai][emocion][feliz][habla][Perfecto, vamos todos juntos!]
[if][encontrar_molino][respuesta2][personaje][Jouktai][emocion][duda][habla][Mmm, intentemos de nuevo mas lejos de las casas.]

[escena][Final]
[personaje][Kai][habla][Jouktai, como hace el molino para sacar el agua?]
[personaje][Jouktai][habla][El viento empuja las aspas y mueve la bomba que jala el agua del pozo.]
[personaje][Kai][habla][Entonces ayudemos al viento!]
[minijuego][sopla_el_viento][respuesta1][respuesta2]
[if][sopla_el_viento][respuesta1][personaje][Kai][emocion][feliz][habla][Mira, el agua esta saliendo otra vez!]
[if][sopla_el_viento][respuesta1][personaje][Jouktai][emocion][feliz][habla][Lo logramos. Recuerda cuidar el agua.]
[cambiar_escena][Chp1_end]
